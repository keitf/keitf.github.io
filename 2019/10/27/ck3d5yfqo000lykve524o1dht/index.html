变量和常量

发表于 2019-10-27 更新于 2019-10-31
本文字数： 430 阅读时长 ≈ 1 分钟

2019年10月27日 12时32分35秒

堆和栈的特点：
栈：用来存储值类型的数据
1、是一个上不封顶的结构
2、栈的数据交换方式是后进先出
3、程序员不管怎么编程都无法回收数据（析构）

堆：用来存储引用类型的数据
1、可以任意开辟空间的区域
2、必须有程序员去管理的一块内存空间。
.Net Framework —-托管

变量：
概念：变量是程序员可以根据自己的需求任意改变值的一块内存空间
语法：
1、先声明，在赋值。先声明，在赋值。
数据类型 变量名 = 值
2、先声明，在赋值。
变量名;
变量名 = 值;
变量名：
1、首字母必须是_或者是大小写字母。
2、其余字母是：_、大小写字母、数字。
3、不能和关键字冲突
4、变量名必须有意义
name/age/person/fatherName/motherName/height/….
xingMing/renMing/baBaMing/
5、一般采用驼峰命名法。

常量】
概念：常量是程序员无论如何不能改变其值的一块内存空间
语法：const
const int x = 100；
